# CookBook - Online Food Ordering Website

## 📌 Project Overview

CookBook is a responsive food ordering web application that allows users to explore restaurants, browse menus, add food items to a cart, place orders, and track delivery status. The project provides an interactive and user-friendly interface inspired by modern food delivery platforms.

The application is built using **HTML, CSS, and JavaScript** and demonstrates front-end development concepts such as dynamic content rendering, cart management, filtering, searching, and order tracking.

---

## Features

### Restaurant Search

* Search restaurants by name, cuisine, or dish.
* Instant filtering while typing.

### Cuisine Filtering

* Filter restaurants based on cuisine categories:

  * South Indian
  * North Indian
  * Chinese
  * Biryani
  * Pizza
  * Desserts
  * Healthy
  * Mexican
  * Seafood
  * Mediterranean
  * Burgers

### Restaurant Listings

* Display restaurant details including:

  * Restaurant Name
  * Cuisine Type
  * Ratings
  * Estimated Delivery Time

### Menu Management

* View restaurant menus.
* Display food descriptions and prices.
* Vegetarian and non-vegetarian indicators.

### Shopping Cart

* Add food items to cart.
* Increase or decrease item quantity.
* Automatic subtotal calculation.
* Delivery fee and tax calculation.
* Cart item counter.

### Checkout System

* Customer information form.
* Phone number validation.
* Delivery address collection.
* Multiple payment methods:

  * Cash on Delivery
  * UPI
  * Card

### Order Tracking

* Real-time order progress simulation.
* Stages include:

  * Order Placed
  * Preparing Food
  * Out for Delivery
  * Delivered

### Notifications

* Toast notifications for:

  * Items added to cart
  * Order status updates
  * Validation messages

### Responsive Design

* Mobile-friendly layout.
* Responsive restaurant grid.
* Modal-based menu display.
* Cart drawer interface.
